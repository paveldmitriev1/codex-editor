# Агент Верности (fidelity_agent) — план

**Цель Pavel-а:** «голосовые надиктовки соотносить к главам и сверять с теми чатами в которых главы были написаны… анализировать и выдавать заключение на сколько ушли от оригинала».

**Принцип:** ничего не теряется. Каждая глава имеет fidelity-score 0-100. Низкий score = драфт ушёл от первоисточника, есть мысли которые система не записала. Pavel смотрит отчёт и решает что вернуть.

## Что агент проверяет

Для каждой канонической главы (например Книга III, Гл 1 «Сущности и Паразиты»):

| Источник | Где | Что значит |
|---|---|---|
| **Голос Pavel-а** (PRIMARY) | `sources/voice-extracts/` + новые надиктовки | прямая речь, не пропускать через AI-перевод |
| **Claude.ai чаты** | `~/Desktop/data-1c39ff64-…/conversations.json` + `projects/*.json` | разговоры где Pavel диктовал главу или её куски |
| **Внешние источники канона** | `sources/book-XX/_external/` | `!0001`, `Последнее Предупреждение`, и т.п. |
| **Текущий драфт главы** (если есть) | `chapters/book-XX/book-XX-ch-YY/main.md` | то что сейчас в работе |

## Что выдаёт

JSON-отчёт + читаемый Markdown:

```
book-03 / Гл.1 «Сущности и Паразиты Сознания» · fidelity 47/100 ⚠

ОСНОВНЫЕ ИДЕИ ИЗ ИСТОЧНИКОВ (5):
  ✓ Эго как оккупированная территория          [есть в драфте]
  ✓ Паразиты питаются болью и страхом           [есть в драфте]
  ⚠ Раздробили целостное восприятие на «осколки» [упомянуто, размыто]
  ✗ Паразиты разрушили связь отец/сын            [НЕТ в драфте — потеряно]
  ✗ Извращённая святость через страдание         [НЕТ в драфте — потеряно]

ДРЕЙФ:
  • Драфт добавил тему «нейрохимии» — её нет в источниках. РИСК.
  • Использован термин «эгрегор» — Pavel говорил «сущность». СБОЙ голоса.
  • Тон стал учебно-научным. Pavel диктовал в режиме откровения.

РЕКОМЕНДАЦИЯ: вернуть 3 потерянные идеи; убрать нейрохимию; восстановить тон.
```

## Архитектура

```
scripts/fidelity_agent.py
  ├── extract_voice_topics()    — читает все voice-extracts, тегирует
  ├── search_claude_archive()   — FTS5 индекс по conversations.json
  ├── extract_canonical_ideas() — Claude API: «выдели 5-10 главных мыслей из этих источников»
  ├── check_chapter_draft()     — Claude API: «для каждой идеи: есть/частично/нет»
  ├── detect_drift()            — Claude API: «что в драфте есть лишнего / не из источников»
  ├── score()                   — 0-100 fidelity
  └── render_report()           — JSON + Markdown в reports/fidelity/<chapter>.md
```

**API:** Claude через ваш OAuth-proxy (`http://localhost:8787`). Модель Opus 4.7 + extended thinking, бюджет 8K tokens на главу. Стоимость по Claude Max плану — минимальная.

**Запуск:**
```bash
python3 ~/Desktop/Codex2/scripts/fidelity_agent.py --chapter book-03-ch-01
python3 ~/Desktop/Codex2/scripts/fidelity_agent.py --book book-03           # все главы книги
python3 ~/Desktop/Codex2/scripts/fidelity_agent.py --all                    # весь Кодекс
```

**В UI:** на каждой главе бейдж `fidelity 47/100 ⚠` (рядом с 📎 и 🎙). Клик → разворот показывает отчёт.

## Зависимости

Нужно:
1. **OAuth proxy работает** (`http://localhost:8787` — уже стоит, проверено)
2. **FTS5 индекс Claude.ai архива** — есть в старом v1 (`Codex/app/search-index.sqlite`, 30 МБ). Скопировать в `Codex2/.codex/` или построить заново
3. **Расшифровка `.docx`** — для чтения источников. Простой вариант: `python-docx` (pip install) или stdlib zipfile+xml. Стdlib работает но грязно.

## Объём работы (оценка)

- **Базовая версия** (один-главу-в-консоль, без UI): 1.5-2 часа
- **+ Полный обход книги с прогресс-баром**: +30 мин
- **+ Интеграция в UI (бейджи + разворот отчёта)**: +1 час
